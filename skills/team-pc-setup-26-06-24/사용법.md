# team-pc-setup — 사용법

## 뭐 하는 스킬인가요?
팀원의 새 PC(윈도우/맥)에 **팀 표준 Claude 작업환경**을 한 번에 깔아주는 마스터입니다.
개인 정보(특정 IP·계정·메인서버·채널·텔레그램 봇)는 전혀 들어있지 않습니다.

## 무엇이 설치되나요?
- 팀 코딩 규칙 `rules/` + 팀 공용 `CLAUDE.md`
- 터미널 세션메뉴 + statusline (claude-term-ux)
- 승인/완료 OS 토스트 알림 (claude-notify-team, 로컬만)
- 탐색기 보기 / 파인더 정렬 고정 (file-manager-view)

## 어떻게 쓰나요? (팀원용)
저장소 **README.md** 에 적힌 프롬프트를 그대로
Claude(커서 터미널)에 붙여넣으면 됩니다. 그러면 Claude가 번들 설치 → 알림/터미널 세팅 →
파일관리자 보기 설정까지 단계별로 안내합니다.

## 진행 방식
- 각 단계 시작 전 **무엇을 할지 설명**하고 확인받습니다.
- 이미 돼 있으면 "건너뛸까요?" 물어봅니다.
- 한 단계가 **검증을 통과해야** 다음으로 넘어갑니다.
- 완전 자동이 아니라, 보기 선택처럼 사람이 필요한 부분만 확인받는 **가이드형**입니다.
