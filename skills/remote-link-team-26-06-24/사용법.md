# remote-link-team — 사용법

## 뭐 하는 스킬인가요?
**본인 윈도우 PC에서 본인 맥에** 무비번으로 붙는 세팅입니다(`ssh mac` + `M:` 드라이브).
남의 맥이 아니라 **각자 자기 맥**에 붙고, 본인 Tailscale 계정으로 묶습니다. 개인정보 하드코딩 없음.

## 준비
- 내 맥 + 내 윈도우 PC가 **같은 Tailscale 계정**에 로그인돼 있어야 합니다(둘 다 켜짐).

## 순서 (2대에서)
1. **맥에서 먼저** — 작업서버 켜기 + 내 값 확인
   ```
   zsh "$HOME/.claude/skills/remote-link-team-26-06-24/scripts/setup-mac-server.zsh"
   ```
   끝에 나오는 **MacUser(맥 사용자)** 와 **MacIp(맥 Tailscale IP)** 를 받아 적으세요.

2. **윈도우에서** — 그 값으로 조종석 세팅(관리자 PowerShell)
   ```
   powershell -NoProfile -ExecutionPolicy Bypass -File "$env:USERPROFILE\.claude\skills\remote-link-team-26-06-24\scripts\bootstrap-cockpit-to-mac.ps1" -MacIp <맥IP> -MacUser <맥사용자>
   ```

## 다 되면
- `ssh mac` → 내 맥에 접속(거기서 claude 실행 가능)
- `M:` 드라이브 → 내 맥 홈 폴더

## 안 되면
- 맥 22 못 닿음 → 두 기기 같은 Tailscale 계정인지 / 1번을 돌렸는지 확인
- 마운트 실패 → 맥 로그인 비번 재확인, `cmdkey /delete:<맥IP>` 후 재시도
- team-pc-setup 마스터로 들어오면 Claude가 위 순서를 단계별로 안내합니다.
