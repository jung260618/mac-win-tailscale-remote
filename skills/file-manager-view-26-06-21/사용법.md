# file-manager-view — 사용법

## 뭐 하는 스킬인가요?
파일 관리자(윈도우 탐색기 / 맥 파인더) 폴더 보기·정렬을 **모든 폴더에 항상 똑같이** 고정해 줍니다.
- **윈도우**: 폴더마다 바뀌는 아이콘 크기·정렬을 한 가지(예: 큰 아이콘)로 통일.
- **맥**: 정렬을 "생성일순 자동 유지"로 모든 위치에 빠짐없이 박음(새 파일 넣어도 자동 정렬).

`explorer-folder-view`(윈도우)와 `finder-sort-cleanup`(맥)을 하나로 합친 스킬입니다.

## 어떻게 부르나요?
그냥 말로:
- (윈도우) "폴더 보기 큰 아이콘으로 고정", "폴더마다 보기 바뀌는 거 고정"
- (맥) "파인더 정리해줘", "정렬 풀렸어", "파인더 생성일순으로"

## 그러면 어떻게 되나요?
1. 무엇을 하는지 먼저 설명하고
2. (맥) 범위를 물어보고 / (윈도우) 원하는 보기를 한 폴더에 직접 맞추게 한 뒤
3. 확인하면 적용합니다. (확인 전엔 실행 안 함)

## 주의
- **윈도우**: 레지스트리 값은 추측하지 않고, 사용자가 만든 보기를 캡처해서 전체에 복제합니다. 적용 전 자동 백업(`FolderView_Backup_*.reg`).
- **맥**: 적용 시 폴더 **창 크기·아이콘 위치**가 기본값으로 리셋될 수 있어요(정렬 자동화 비용). 네트워크 볼륨은 느려서 기본 제외(원하면 포함).
- 직접 실행: 윈도우 `setup.ps1 -Backup/-FindUserBag/-Apply`, 맥 `setup.sh home` 또는 `setup.sh <폴더>`.
