# 사용가이드 — claude-term-ux
(클로드 터미널 화면 세팅)
갱신일: 2026-06-17

## 이게 뭐 하는 스킬?
새 PC나 환경 재설치 후, 클로드
터미널을 편하게 쓰는 두 가지를
한 번에 깔아줍니다.

## 언제 쓰나 (이렇게 말하면 켜짐)
- "클로드 터미널 세팅"
- "세션 메뉴 다시 깔아줘"
- "하단 표시줄(statusline) 복원"
- "다른 PC에 하단바 세팅"

## 주요 기능
- `claude`만 치면 최근 세션 20개를
  2줄 카드로 띄워 골라서 이어가기
  · 1줄: 주황 번호 + 제목(없으면
    대화 요약 자동생성) + 경과시간
  · 2줄: ↳ 마지막 보낸 명령어 + 폴더
  · 길면 터미널 폭에 맞춰 자동 줄바꿈
- 원격 ssh 접속해서 `claude` 쳐도
  똑같이 메뉴가 뜸(한글 안 깨짐)
- **맥 로컬 터미널**에서 `claude` 쳐도
  똑같은 메뉴가 뜸(zsh, 추가설치 없음)
- 하단 표시줄: 세션이름·컨텍스트·
  사용량(5h/7d)·모델을 보여줌(윈도우)
- 터미널/분할창 폭에 맞춰 자동 줄바꿈
- 모든 단계 멱등(이미 돼 있으면 건너뜀)

## 입력 → 결과
입력 : "터미널 세팅해줘"
결과 : 세션 메뉴 래퍼 + 하단 표시줄
       설치 완료

## 설치·주의
- 세션 메뉴: 윈도우(PowerShell)+맥(zsh)
  양쪽. statusline 은 윈도우 전용(node).
- 맥은 `install.sh`, 윈도우는 `install.ps1`
  이 자동으로 깔아줌. 새 터미널부터 적용.
- 맥에서 제목 없는 세션 요약을 쓰려면
  `claude` `/login` 필요(안 돼도 첫 발화로
  대체, 메뉴는 정상).
- 어떤 사용자명에서도 작동.
- 자세한 규칙은 같은 폴더 SKILL.md 참고.
