# claude-notify-team — 사용법

## 뭐 하는 스킬인가요?
Claude Code가 **승인을 기다리거나 작업을 끝냈을 때** 화면을 막는 대화상자 대신
**OS 토스트 알림**(맥 알림센터 / 윈도우 토스트)으로 알려줍니다. 팀 공용본이라 개인 PC 연결과 무관하게 동작합니다.

## 어떻게 부르나요?
보통은 `team-pc-setup` 마스터가 자동으로 깔아줍니다. 알림만 다시 세팅하려면:
- 맥: `bash "$HOME/.claude/skills/claude-notify-team-26-06-24/scripts/setup.sh"`
- 윈도우: `powershell -NoProfile -ExecutionPolicy Bypass -File "$env:USERPROFILE\.claude\skills\claude-notify-team-26-06-24\scripts\setup.ps1"`

## 확인
세팅이 끝나면 테스트 토스트가 한 번 뜹니다. 안 뜨면:
- 맥: 시스템 설정 → 알림 → 터미널/Cursor 알림 허용 확인.
- 윈도우: 집중 지원(Focus assist)이 꺼져 있는지, 알림이 켜져 있는지 확인.
